discrption:- 
Paw-fect Match is a minimum viable mobile application designed to connect pet owners by enabling them to create profiles for themselves and their pets. The goal of this project is to demonstrate fundamental skills in React Native development, REST API integration, JWT authentication, and backend architecture.
This project focuses on building a clean UI, implementing secure authentication, creating structured backend APIs, and maintaining a scalable codebase.

Deployment

Backend: Render    (The backend was deployed on Render because it allowed for faster setup and deployment compared to AWS. While attempting to use AWS, my account was not being activated properly, possibly due to a configuration or verification issue. To avoid delays and ensure timely project completion, I chose Render as it provides a quick, reliable, and developer-friendly deployment platform

Mobile App: Expo (with compiled APK for direct installation)


Setup:-

1) Clone the Repository
   
3) Backend Setup ( cd backend ) then npm install
   
5) Create env file PORT=5000 , MONGO_URI, JWT_SECRET, EMAIL, EMAIL_PASS, CLOUD_NAME, CLOUD_KEY, CLOUD_SECRET
   
7) then Run the server: npm start
   
9) (cd frontend) npm install then npm start0

11) eas build -p android --profile preview ( generate APK )
